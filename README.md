 Room Allocation System

📌 Project Overview
The Room Allocation System is a web-based application developed using PHP and MySQL. It automates classroom allocation based on student strength and room capacity, reducing manual effort and improving accuracy.

🚀 Features
- Admin Login
- Department Management
- Class Management
- Room Management
- Automatic Room Allocation
- Manual Room Allocation
- View Allocation Details

🛠️ Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap

💻 Installation
1. Install XAMPP.
2. Copy the project folder into `htdocs`.
3. Import `room_allotment_system__1_.sql` using phpMyAdmin.
4. Start Apache and MySQL.
5. Open the project in your browser.
